## 2. 메인 유저 플로우

### Flow 1: 로그인 → 제품 탐색 → 구매 완료 (핵심 플로우)

```
┌─────────┐  사용자 아이콘  ┌─────────┐   가입 완료    ┌─────────┐
│   S01   │ ────────────> │   S07   │ ────────────> │   S01   │
│  랜딩   │               │  로그인  │  (Google OAuth) │  랜딩   │
└─────────┘               └─────────┘               └────┬────┘
                                                          │
                                                    CTA "컬렉션 보기"
                                                          │
                                                          ▼
┌─────────┐   "주문 완료"   ┌─────────┐  "주문하기"   ┌─────────┐
│   S06   │ <───────────── │   S05   │ <──────────── │   S04   │
│주문완료 │                │ 체크아웃 │               │ 장바구니 │
└─────────┘                └─────────┘               └────┬────┘
                                                          │
                                              "장바구니 담기" (S03에서)
                                                          │
┌─────────┐   카드 클릭    ┌─────────┐               ┌────┴────┐
│   S02   │ ────────────> │   S03   │ ────────────> │   S04   │
│ 제품목록 │               │ 제품상세 │  장바구니 아이콘  │ 장바구니 │
└─────────┘               └─────────┘               └─────────┘
```

**단계별 상세:**

1. **S01 → S07:** 사용자 아이콘 클릭 → Google 로그인
2. **S07 → S01:** 로그인 완료 → 홈으로 이동
3. **S01 → S02:** 히어로 CTA "컬렉션 보기" 클릭
4. **S02 → S03:** 제품 카드 클릭 (필터/정렬 후 선택 가능)
5. **S03 → S04:** 색상·수량 선택 후 "장바구니 담기" → 헤더 장바구니 아이콘 클릭
6. **S04 → S05:** 수량 확인/조절 후 "주문하기" 클릭
7. **S05 → S06:** 배송 정보 입력 + 결제 수단 선택 → "주문 완료" 클릭

---

### Flow 2: 비로그인 → 장바구니 담기 → 로그인 유도

```
┌─────────┐   카드 클릭    ┌─────────┐  "장바구니 담기"  ┌─────────┐
│   S02   │ ────────────> │   S03   │ ──────────────> │   S07   │
│ 제품목록 │               │ 제품상세 │  (비로그인 감지)   │  로그인  │
└─────────┘               └─────────┘                 └────┬────┘
                                                           │
                                                     로그인 성공
                                                           │
                                                           ▼
                                                      ┌─────────┐
                                                      │   S03   │ (원래 제품 페이지로 복귀)
                                                      │ 제품상세 │
                                                      └─────────┘
```

---

### Flow 3: 베스트셀러에서 바로 구매

```
┌─────────┐  베스트셀러 클릭  ┌─────────┐  장바구니 담기  ┌─────────┐
│   S01   │ ──────────────> │   S03   │ ────────────> │   S04   │ → S05 → S06
│  랜딩   │                 │ 제품상세 │               │ 장바구니 │
└─────────┘                 └─────────┘               └─────────┘
```

---

### Flow 4: 제품 목록 탐색 (필터 & 정렬)

```
┌─────────┐                    ┌─────────┐
│   S02   │ ── 카테고리 필터 ──>│   S02   │  (필터링된 목록)
│ 제품목록 │ ── 정렬 변경 ────> │ 제품목록 │  (정렬된 목록)
└────┬────┘                    └────┬────┘
     │ 카드 클릭                     │ 카드 클릭
     ▼                              ▼
┌─────────┐   "장바구니 담기"  ┌─────────┐
│   S03   │ ── + 뒤로가기 ──> │   S02   │  (다른 제품 계속 탐색)
│ 제품상세 │                   │ 제품목록 │
└─────────┘                   └─────────┘
```

---

### Flow 5: 장바구니 수정

```
┌──────────────────────────────────────────┐
│                S04 장바구니               │
│                                          │
│  ┌─ 수량 변경 (+/-) → 소계/총액 갱신     │
│  ├─ 아이템 삭제 → 목록에서 제거 + 총액 갱신│
│  ├─ 모두 삭제 → 빈 장바구니 안내 표시     │
│  └─ "주문하기" → S05 체크아웃             │
│                                          │
│  [빈 장바구니] → "쇼핑하러 가기" → S02    │
└──────────────────────────────────────────┘
```

---

### Flow 6: 주문 완료 후 재탐색

```
┌─────────┐  "쇼핑 계속하기"  ┌─────────┐
│   S06   │ ──────────────> │   S02   │ → Flow 1 반복
│주문완료 │  (장바구니 비움)  │ 제품목록 │
└─────────┘                 └─────────┘
```

---

### Flow 7: 마이페이지 (주문 내역 확인)

```
┌─────────┐  사용자 아이콘  ┌─────────┐
│ 어디서든 │ ────────────> │   S09   │
│         │  (로그인 상태)  │ 마이페이지│
└─────────┘               └────┬────┘
                               │
                    ┌──────────┼──────────┐
                    ▼                     ▼
              주문 내역 목록          프로필 정보
              (주문 클릭 시           (이름, 이메일,
               상세 펼침)              연락처)
```

## 3. 네비게이션 맵

```
                    ┌────────────────────────────────────┐
                    │          헤더 (C01/C02)             │
                    │                                    │
                    │  로고    메뉴        아이콘          │
                    │   │    ┌──┴──┐    ┌───┴───┐       │
                    │   │    │     │    │       │       │
                    │   ▼    ▼     ▼    ▼       ▼       │
                    │  S01  S02  S01#  S04   S07/S09    │
                    │  홈  컬렉션 브랜드 장바구니 로그인/마이│
                    └────────────────────────────────────┘

  S01# = 랜딩 페이지 내 브랜드 소개 섹션으로 스크롤
  S07/S09 = 비로그인 시 S07(로그인), 로그인 시 S09(마이페이지)
```

| 어디서든 | S04 `/cart` | 장바구니 아이콘 클릭 |

| 어디서든 | S09 `/mypage` | 사용자 아이콘 클릭 (로그인) |
| S01 | S02 | 히어로 CTA "컬렉션 보기" |
| S01 | S03 | 베스트셀러 카드 클릭 |
| S02 | S03 | 제품 카드 클릭 |
| S03 | S07 | "장바구니 담기" (비로그인 시) |
| S03 | S04 | "장바구니 담기" 후 장바구니 아이콘 클릭 |
| S03 | S03 (다른 제품) | 관련 제품 추천 카드 클릭 |

## 5. 스크린별 상태 정리

| 스크린 | 상태 | 설명 |
|--------|------|------|
| S02 제품 목록 | 기본 | 전체 제품 표시 |
| | 필터 적용 | 선택 카테고리만 표시 |
| | 정렬 적용 | 선택 기준으로 정렬 |
| | 결과 없음 | 필터 조합에 해당 제품이 없을 때 안내 |
| S03 제품 상세 | 기본 | 첫 번째 색상, 수량 1 선택 상태 |
| | 색상 변경 | 다른 컬러 스와치 선택 시 이미지 전환 |
| | 장바구니 추가 완료 | 토스트 또는 뱃지 업데이트 피드백 |
| S04 장바구니 | 아이템 있음 | 아이템 목록 + 주문 요약 |
| | 비어있음 | 안내 문구 + "쇼핑하러 가기" 버튼 |
| S05 체크아웃 | 입력 중 | 폼 작성 진행 (프로필 정보 자동 입력) |
| | 유효성 에러 | 필수 필드 미입력 시 에러 메시지 |
| | 제출 가능 | 모든 필수 값 입력 완료, 버튼 활성화 |
| S06 주문 완료 | 기본 | 주문 번호 + 내역 요약 표시 |
| S07 로그인 | 기본 | Google 로그인 버튼 |
| | 에러 | 로그인 실패 메시지 |
| | 유효성 에러 | 이메일 형식, 비밀번호 길이 등 |
| S09 마이페이지 | 주문 내역 | 주문 목록 (접기/펼치기) |
| | 프로필 | 회원 정보 표시 |
